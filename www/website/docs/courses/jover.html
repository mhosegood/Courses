Java developer courses from First Alternative

Our Java courses are designed to provide training for today's development environment, with emphasis on E-Commerce development. We offer a fully-configured development environment, using the most up-to-date E-Commerce solutions (BEA's Weblogic server, and iPlanet web server, Enterprise Edition).

We start with Java Programming, a 5-day in-depth course giving developers already experienced in C, C++ or similar languages a flying start in Java.

Two further courses are then available:-
- J2EE Web Application Development with BEA WebLogic 6.0/iPlanet Web Server
- Enterprise Java Beans Development with BEA WebLogic 6.0

Please select this link for a list of all our scheduled courses.

 

Java Programming

This 5-day course focuses on the development Java applications using the Java 2 specification. The course concentrates on client side technologies, but includes an introduction to server side technologies (JDBC, J2EE and EJB).
It is suitable for experienced developers with little or no previous Java knowledge.

J2EE Web Application Development with BEA WebLogic 6.0/iPlanet Web Server

This 5-day course focuses on web application development with J2EE using BEA's latest Web Server - WebLogic Server 6.0 and the iPlanet web server, Enterprise Edition. The course concentates on the core J2EE technologies including an introduction to EJBs.
The course is based on the Sun Solaris platform but is suitable for any other platform employing the same technologies.

Enterprise Java Beans Development with BEA WebLogic 6.0

This course focuses on developing Web Applications with Java 2 Enterprise Java Beans using BEA's latest Web Server - WebLogic Server 6.0 The course concentates on EJBs, but also shows how to integrate EJBs with servlets and JSPs.
The course is based on the Sun Solaris platform but is suitable for any other platform employing the same technologies.
