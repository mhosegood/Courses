Overview of our Perl Courses

This page includes a short description of each of the public courses we offer on Perl, and links to full descriptions of each of them. Please select this link for a list of all our scheduled courses.

 

Perl is a language for easily manipulating text, files and processes. Perl provides a more concise and readable way to do many jobs that were formerly accomplished (with difficulty) by programming in the C language or one of the shells. Even though Perl is not yet a standard part of UNIX, it is likely to be available at any UNIX site. And if it isn't, users can get it and install it easily and free of charge.

If you're going to be writing Perl Programs, but aren't an experienced programmer, start with the Perl Basics course and carry on to Perl Programming

If you're already an experienced programmer, start with Perl Programming

Perl Basics

Our Perl Basics course is a gentle introduction to the language for those who have only limited (or no) previous programming experience in a language such as C or awk. This one-day course will give you the foundation to progress on to our Perl Programming course.

Perl Programming

Our Perl Programming course covers all the major building blocks of the language. Prior programming knowledge, in a language such as C, is a prerequisite if you want to start straight into this four-day course. Otherwise, please attend Perl Basics first on the day before.

Upon completion of the Perl Programming course, you will be equipped to use Perl for many data manipulation and Web interaction tasks. If you have three or more staff to train, may we suggest you visit our custom department for information about extra modules and on-site training?
