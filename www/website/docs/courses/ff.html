TOPICS COVERED ON THE COURSE

The basics
Getting FrameMaker started. What is a FrameMaker document?. What are the capabilities of FrameMaker? Understanding document templates. Opening a document template to prepare a new document.

Navigation
A tour around the document to show how menus and dialogue boxes are invoked and used, including the Formatting Ruler and Quick Access Bar.

Basic text manipulation
How to enter and edit text. Text selection, including the use of the mouse to Copy, Cut and Paste. Finding text. Find and replace. Spelling Checker and Thesaurus.

Paragraphs
Paragraph Formats Using the existing paragraph formats in a document or template to apply formats to text. Creating your own paragraph designs with The Paragraph Designer. Including fonts and tabs. Pagination properties. Run-in and side-heads. Automatic paragraph numbering (includes multiple numbered paragraphs).

Overrides
Overriding text formats with the Character Designer. Creating your own character formats.

Graphics
Adding graphics to your document. Ensuring that they are correctly inserted in the flow of text. A comprehensive study of the graphics Tools Palette. The use of anchored frames and runaround properties.

Tables
How to use FrameMaker's powerful tables facility. Create and modify your own tables. Using Table formats and creating format entries.

Footnotes and Change Bars
Creating and modifying document footnotes and change bars.

Page Layout
Creating a Page Layout design. Add the professional finishing touches to a document. Set up and usage of Master Pages. Single/double sided documents. Creating and applying custom Master Pages.

Reference pages
The use of graphics on reference pages and using them with paragraph formats.

Reference - 5.5.6 and 6 differences
The section provides a reference for the major differences between versions 5.5.6 and 6 of FrameMaker.

Getting started with Quadralay Webworks Publisher
This product is provided with FrameMaker version 6 and provides enhanced facilities for web page and site creation.

HELP
Using the extensive FrameMaker HELP facilities. Reference material or guidelines for any problems you may encounter.

Custom document creation
The course concludes with the creation of a complete custom document, employing all the facilities of FrameMaker previously covered, with a few more hints and tips.
