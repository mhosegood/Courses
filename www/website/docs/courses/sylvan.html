A testing time for all Solaris and Linux Administrators

This page describes certification for the Solaris or Linux Administrator

 

Solaris

As with many other operating systems and applications, a certification path is available for the Solaris administrator. The certification is achieved by passing exams run independently of Sun by the Sylvan Prometric organisation, who have many testing centres around the UK.

Contrary to rumours, and as several of our clients will verify, you don't have to attend Sun's courses to sit (and pass) these exams!

Sylvan Prometric offer two levels of examination, to be taken in the following sequence:

  • The Certified Solaris Administrator Program for Solaris 8, or earlier (This consists of two exams, each of 90 minutes duration.)
  • The Certified Network Administrator Program for Solaris 8, or earlier (A further 90-minute exam.)

Both exams may be taken at any Sylvan Prometric testing centre. Before sitting these exams we recommend that your Solaris experience matches that covered by the following courses:

  • Introduction to Solaris (course code SI)
  • Solaris for Users/Administrators (course code SA)
  • Solaris 8 System Administration (Part 1) (course code S2A1)
  • Solaris 8 System Administration (Part 2) (course code S2A2)
  • Solaris 8 Advanced System Administration (course code S2ASA)

As a delegate, you should also have a fair degree of practical experience in readiness for the broad-based Solaris and networking topics you may meet on the question papers. You should also be aware that you may be questioned on Solaris-related products such as Java stations and Auto-clients. However, there is no obligation to attend any courses; if you feel confident you have the skills, just sign up! Call Sylvan Prometric on (0800) 592873 or (0208) 607 9090 for details of your local testing station. Or visit their website www.sylvanprometric.com

Linux Certification

Our courses are designed to help achieve the following two certifications:

  • Linux Professional Institute (LPI)
  • RedHat Certified Engineer.

LPI is a vendor-neutral certification and is taken through the VUE testing organisation, for which there are many testing centres around the country. For full information about the LPI exams, look at www.vue.com. This web site contains not only registration details, but also telephone numbers of regional offices.

RedHat Linux certification is run by RedHat themselves. It takes a day, and covers practical tests as well as a multi-choice question exam. Full details are available at www.redhat.co.uk. Your knowledge should be up to a level as taught in the following courses:

  • Introduction to Linux (course code SI)
  • Linux for Users/Administrators (course code SA)
  • Linux System Administration (Part 1) (course code S2A1)
  • Linux System Administration (Part 2) (course code S2A2)
  • Linux Advanced System Administration (course code S2ASA)

Ideally, you should combine the above with about 6 months of practical experience, and study the exam guides carefully. Remember that you can book all the above via our LI2 package, giving you 13 days of instructor-led training for a price that leaves other training companies standing!
