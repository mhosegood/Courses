TOPICS COVERED ON THE COURSE:

Perl Programming Introduction
What is Perl? Using and obtaining perl.

More about the Perl Environment
Integrating Perl with your computer. Unix and Linux - Path, chmod and the #! line. Windows NT, 98 and 95 - Associating files with Perl. Running Perl on the PC under MS-DOS.

Perl Fundamentals
First Perl Program. Comments and executable statements. Print statements. Variables and assignments. Calculations. Reading Data. Strings v numbers. String operators.

Conditionals and loops
The if statement.. The while statement. Shorthand assignment operators. Single statement conditionals. Unless. Conditional operators. The until loop. The for loop. Single statement loops. Breaking a loop. Labels.

Initial String Handling
String handling functions and operators. Quoted strings. Exact string comparisons. Regular expression matching.

File Handling
File Handles. The open and close functions. Reading and writing files.

Formatted Printing.
Formatting variables and expressions. Constant text. sprintf.

Lists
What is a list? Creating a list. Referencing elements in a list. Changing a list. The length of a list. List and scalar context. Functions that operate on lists. List slices.

Subroutines
What and Why of subroutines. Calling a subroutine. Loading a subroutine from another file. Passing Parameters. Writing Subroutines. Returning values.

Variable scope.
Local, my and package and special variables. Loading a class and using objects. Instances. Command line parameters -. Informational variables. Behavious changing variables. $_ - the default input and pattern matching variable. More command line options.

Hashes
What is a hash? Setting up a hash. Keys and values. Accessing a hash. Sorting. cmp and operators. Programming techniques for hashes. Deleting hash elements. Special hashes.

Character Strings
Extracting information from a match. Assigning to a list. Matching strings. Pre and Post match strings. Modifiers. Comments within a regular expression. Alternative delimiters. Substitution. Substitute and execute. Regular expression efficiency. The tr function. Handing binary data - read, unpack etc.

Creating your own classes
using a class. Background to instance variables - pointers and references. new - the constructor. blessing. Other methods in a class. Using more complex structures to store data in a class. Extended capabilities by using hashes.

More about files
Further file enquiries. globbing. stat. File locking. Random access files. Accessing the structure of the file system. Including data in your programs. write and format. Handling dates and time File status operators and stat. System commands. time, utime and other sources of time data.

Libraries and Resources
Common modules such as English, Env, time and Config. Strict, cwd, and Carp. Command line handlers. Other standard modules. The CPAN what it is and how to use it. Flat databases.

Perl on the Web
The HTML form. Input from the form into a Perl program - URL encoding. Output from Perl back to the browser - headers, etc. Overview of what you can do with CGI.
