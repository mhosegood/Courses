Overview of our Solaris Courses

First Alternative's bread and butter are our range of Solaris courses. For nearly 10 years we have been providing the Solaris community (both in the UK and Europe) with an alternative to the manufacturer's courses.

We offer a broad range designed at getting the most out of Solaris, no matter whether your role be user, administrator or developer.

For those who complete our entire range from Introduction to Advanced administration, the path is clear to sit the Solaris certification exams with the independent Sylvan Prometric organisation. All the skills you need are taught here, from basic system operation, through Solaris utilities, to shell programming and many advanced networking and admin topics.

Our Solaris courses are suitable training for Solaris Certification

Introduction to Solaris

Our Introduction to Solaris course covers Solaris fundamentals, but in a practical and thorough way. You'll feel confident and eager to learn more, and will have all the skills necessary to handle day-to-day Solaris sessions. Many people comment that this is a "well-rounded" course, and sets them up perfectly to progress further. Logging in, file manipulation, directory skills, documentation, vi editor, file permissions, processes, shell command-line skills (IO control, history, aliases, etc. - even using quotes ! uggh!) Want more? Utilities overview, and CDE windows system are also covered, plus to finish off, a network services overview. A truly comprehensive course which provides the best possible springboard into your Solaris career.

Solaris for Users/Administrators

If you administrate Solaris, or look after applications support (or a hundred other similar technical roles!) then our Solaris for Users/Administrators course provides essential skills to get the absolute most out of the Solaris system. The two main areas covered are Solaris utilities (the nawk, sed, grep and sort variety, plus many others including tar, dd and find; Oh, and differentail commands such as comm and diff) and Bourne/Korn shell programming. These sections will give you the skills required to write (and understand= !) complex shell command procedures to automate many tasks such as reports, backups, adding users, data manipulation, system monitoring, and so much more. You'll also learn the more intricate and powerful aspects of the vi editor (Not too long on this though!) plus the print spooling mecahnism. A real power-users course designed to make the most of Solaris.

Solaris 8 Administration - Part 1

Essential skills for successful and secure systems administration. Solaris 8 System Administration (Part 1), the first in a series of three admin courses, concentrates on the important areas common to all workstations, such as Solaris installation, software= management the admin GUI tools, Stsratup and Shutdown, File system management and security, taking backups AND restoring (even a complete system!), plus adding printers and disks. Everyone who needs to look after a Solaris system should start here.

Solaris 8 Administration - Part 2

For those who manage networked Solaris machines, the Solaris 8 System Administration (Part 2) course is as essential as the part 1 and covers all the skills necessary to successfully network your Solaris machines. You will understand how to physically connect them, then look at the IP address scheme and get machines talking. Basic access commands, security control, network processes, and network monitoring are all there. Move on to thoroughly explore the NFS (Network File System) file sharing mechanism (even auto-mounting user home directories!) Learn how to store and manage admin files centrally using the sophisticated NIS+ name service (a full working set up of NIS+ is created - by YOU!) Finally, configure e-mail on your local Solaris network, and learn how to set up Sun's Web Server to ceate your own web site.

Solaris 8 Advanced System Administration

Solaris 8 Advanced System Administration is great 5-day course for those involved in larger networks, and professionals who need to have the broad range of skills to offer in todays Solaris market. Kernel parameters, X- terminals, low-level IP files, in-depth routing, Jump Start and Build servers, OnLine DiskSuite, and many other topics. Full Name service coverage, including NIS and DNS complete set-up, and advanced NIS+ installations make this a course you can't afford to miss!
