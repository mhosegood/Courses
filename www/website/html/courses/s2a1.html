System Administration Essentials

This three-day course is designed to cover the essential administration tasks confronting today's Sun systems administrators. It covers all basic workstation administration functions, with full hands-on experience at all stages. The new Solaris 8 Visual Administration admintool and Adminsuite are also covered. For network administrators, this course should be followed by the Solaris 8 System Administration (Part 2) course (S2A2).

INTENSITY: About 40% of the course is practicals, and 60% lectures.

RESOURCE: Each student will have exclusive use of a Sun workstation for the duration of the course. Each student will be provided with a full set of training notes relating to the course, and quick reference cards to assist with file editing and Unix commands

Printer(s) are provided on this course so that students can learn how to use them, and also take away printed copies of their work.

Tape drives are provided for each student for use during backup and restore exercises.

PREREQUISITES: Knowledge of Solaris up to a level taught in our Introduction to Solaris (SI) course is important, and some previous administration experience is helpful. Please call if you require any advice.

FOLLOW UPS: Solaris 8 System Administration (Part 2) (S2A2), which covers essential network administration, followed by Solaris 8 Advanced System Administration

Our Solaris for Users/Administrators (SA) is another popular course for administrators wishing to gain knowledge of shell programming and Solaris utilities, with special emphasis on the needs of the administrator.

PRACTICALS / TAKEAWAYS: Each student leaves the course with their own set of training notes for the material covered; around 120 pages per day of training.

SOFTWARE VERSIONS: This course is based on Solaris 8 (Solaris 2.8, SunOS 5.8). It is also suitable for customers using Solaris 7, 2.7, and 2.6.

This course is not suitable for other versions of Unix.

 

TOPICS COVERED ON THE COURSE

System Administration Overview
Hardware types. Workstation configurations. Administration functions.

System Build
Planning disk layout. Using the commands available at the monitor prompt (boot, probe-scsi, test-net, etc. Booting from the CD or network to perform the build. Building Solaris from scratch; selecting software options, disk slicing, etc. (Includes Webstart browser based installation including co-packaged software such as Solaris Adminsuite, plus full interactive Solaris build) Post-installation procedures. Building when connected to a network. Overview of automatic and remote installation.

Software Configuration
How to set up and customise a variety of tools, including shells, Common Desktop Environment, manual pages and AnswerBook. Examining, Installing and removing software options.Using admintool and webstart installer to install software. Using the Product Registry. Examination of the major administration GUI tools, including Adminsuite and Solaris Management Console.

Startup and Shutdown
Switch on; boot process and problems. Kernel load, init, inittab and the rc scripts. Adding your own scripts at system boot. Run states.Understanding and changing run states. Halting, suspending and powering off the system. Configuring the eeprom. Power management.

Adding and Maintaining Users
Concepts. Ownership of files, directories and processes. Classes of user. Adding a user manually, via command line tools and graphically.. Password control. Real and effective id. Using admintool and Adminsuite for user management.

File System Maintenance and Security
Utilities for file system management. Timing commands with crontab. Protection mechanisms, including access modes, s and t bits, umask, chown and chmod. Access Control Lists (ACL). File system structure and slicing. The mount command. Managing and Adding swap space. Handling CD and floppy disks. Major file and directory tour. Make room on the file system. File system security. Adminsuite file system functions.

Solaris Groups
Overview. The group file. Group identification of files and directories. The newgrp, groups and chgrp commands.

Back-Up and Restore Utilities
Overview and Preparation. Tape types and capacities. Sample ufsdump commands. Backup strategies. Taking a complete backup. Ufsdump command options. Restoring files, directories and complete file systems. Restoring an entire system. The tar, cpio and mt utilities. Backups versus archives. Overview of Adminsuite backup.

Adding a Printer
Overview. Connection methods. Printer types. Software modifications. Spooling system commands. Administrator control command. Printer classes. Networking a printer. Using Admintool and Adminsuite for printer management.

Adding a Disk
Physical connection. Slicing. Creating a Solaris file system with newfs. Deciding how to mount the new files system; updating /etc/vfstab. Sharing the new file system on the network. Using Adminsuite to perform all the above functions.

Adding a Terminal
Provided as reference material. Overview. Connection methods. Software modifications required. Testing and setting correct terminal type.
