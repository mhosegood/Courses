TOPICS COVERED ON THE COURSE

Advanced Page Layout
Including the creation of special text frame flows. Setting up Master Pages for parallel flows.

Variables
How to use system variables such as automatic page numbers, dates, etc.. Changing variable definitions. Setting up user-defined variables. Using variables in tables.

Cross-References
Inserting a cross reference. Customising existing and setting up your own cross-reference formats. Updating cross references. Dealing with unresolved references.

Text Insets
Importing and updating text insets.

Conditional Text
Viewing a conditional document. Understanding the mechanics of a conditional document. Modifying and setting up condition tags.

Document control tools
Including generating document word count and comparison reports, etc.

Template creation
How to create your own basic and complex template documents. Character and paragraph catalogues. Master Pages. Custom master pages.

Hypertext
The creation of basic Hypertext elements. Alert, gotopage and gotolink. Saving the document in locked (View-only) format. Overview of other Hypertext commands. Save a file as HTML and view in a web browser. Refine HTML mappings. Split a document to multiple web pages. Add links to other web pages.

Table of Contents
Generating a TOC. Formatting a TOC. Using Hypertext links in a TOC. Saving a TOC and using it as a template for future TOCs. Indexes - Preparing a document for index generation. Adding and editing indexmarkers. Marker formats. Generating the index. Formatting the generated index. Using the generated index as a template.

Book Files
What is a book file? Preparing documents for a book. Creating A book file. Including documents. Changing order of documents. Generating TOCand index for a book. How to use running chapter and page numbers in a book. Book-wide commands, drag-and-drop user interface for managing chapters and general files.

Colour
How to use colour in your documents. Colour models. Assigning colours to objects. Creating a new colour. Setting up colour views. Printing colour separations.

PDF Support
How to save documents in PDF format for viewing in Adobe Acrobat and Acroread.
