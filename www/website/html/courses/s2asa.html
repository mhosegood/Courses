S2ASA

System Administration tasks for complex networks

This five-day course is designed to cover the more technically complex and difficult tasks confronting the Solaris/UNIX system and network administrator. All topics will be accompanied by extensive hands-on, with a minimum ratio of one fully-equipped Sun workstation between two students. There will be ample time to put forward your own topics for discussion and analysis.

TOPICS COVERED ON THE COURSE

Kernel Configuration Parameters
What parameters are available to configure. Open files, processes, eliminating device drivers, etc. Examining kernel parameters in the current kernel. Editing /etc/system to enforce new values. Using utilities such as vmstat, sar, etc to monitor and analyse system activity and resource use.

Advanced networking revision
Adding a machine to the network. Connection testing. Implementing network security. Setting up and enabling NFS.

X Windows Technology Explained.
X Windows background and constituent parts. CDE and Open Look. X windows and the network. Adding an X terminal to the network (xdm and setup files; booting; dot files). CDE and dtlogin. Configuring the Chooser.

The Internet Network files.
Background. Address Scheme. Address resolution protocol (ARP). Interface configuration. Using arp and ifconfig. Creating a logical interface. Protocols (IP, TCP, UDP, RPC). Network Administration files. Internetworking Routers and routing configuration. Using in.routed, route and ndd. Internetworking and NIS. Sub-netting a registered address. Multicasting. DHCP and other protocols. IPv6 Overview.

NFS Automounter
Maps and indirect maps. The host map. Modifying maps. Maps and NIS. Automounter Metacharacters. Automounter startup & options.

Adminsuite and Solaris Management Console
Installation. Facilities. Configuration for use with multiple servers.

Naming services
A thorough examination of the major administration Name Services available under Solaris 8. NIS, NIS+ and DNS and how they co-exist and interact.

NIS (Network Information Service)
The original Sun name service, now available in full on Solaris 8, and covered here in detail. Master, Slave server and client set-up. Day-to-day maintenance. Using the netgroup facility. Integration with NIS+ and DNS. Creating your own NIS data.

DNS (Domain Name Service)
Terminology (servers, zones, resources etc.). Setting up DNS Using DNS. Sorting out DNS problems. More complex DNS setup. Administrating DNS.

Advanced NIS+
Review of setting up and administrating a basic namespace. Creating further domains. Adding your own tables. Adding replica servers. Removing and updating credentials. The NIS+ transaction log. Setting up lower-level domains. NIS+ utilities such as nisgrep and nismatch. Setting up your own NIS+ information tables.

Accounting Mechanisms
Login accounting - viewing data, analysis utilities. Process accounting - enabling, examining and analysing data.

Printing via the Network
Setting up the printer host. Setting up the client. Network printer integration. Printers directly connected to the network. Centralising your Solaris 2 printer configuration with Adminsuite facilities. Using the name services with printer details.

Automatic Solaris Installation
Jump Start. Build servers. Copying Solaris to disk. Creating rules and profiles for automatic build. Booting and building from the network interactively and automatically.

Low-level disk maintenance
Partitioning & geometry. Running format. Repairing bad sectors. Surface Analysis. Defect management. SCSI targets explained. Bus termination.

Disk quotas
Enabling quotas. Setting quota limits and time limits; Quota action during normal running. Quota reporting.

On-line disksuite
Overview. Product installation. The metadevice state databases. Creating concatenated and striped metadevices. Creating and adding mirrors. removing mirrors. taking mirrors off-line. UFS logging. Hot Spares. Expanding file systems with growfs. Summary of files and commands. Using the GUI to perform ODS tasks and monitor ODS status.

Network monitoring
Reference Material. snoop, netstat, nfsstat etc.
