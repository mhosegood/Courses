Technical Tip

QUESTION

My (Sun) ID Prom has failed. Can I use my machine? Can I replace the ID Chip with a regularly available chip, or do I have to wait for one to be made?

ANSWER

Yes - you *can* use your machine. But you MAY need to enter its ethernet address and serial number each time you power it up.

Here's an example for a Sparc 1+

  • Power up; when testing, hold stop key and enter "A" to give you the PROM monitor prompt of ">"
  • Switch to new command mode by entering "n" and return
  • In new command mode, enter the following:
      0 0 mkp
      8 0 20 a b c 53001234 mkpl
      ^D^R
    Which sets up an ethernet address and serial number
  • Boot the machine. For example, you might enter
      old-mode
      b sd(0,2,0)
    To boot from the disc as SCSI target 2

For a permanent fix - yes, you can replace the PROM chip by a commercially available PROM; once you have done so, the commands listed above will cause the new settings to be saved. As sources may change, please email Mick Hosegood for current information.

NOTES

  • The ethernet address (8 0 20 a b c) must be unique to each workstation on your network, so if you have (or have had) several systems fail, choose a different address for each! The "8 0 20" means Sun - don't change that; the "a b c" are three hex pair - so you have a huge choice; it would work equally well with "5 3a 12" or "44 55 66", for example
  • The hostid must also be unique to your network. The first four digits are usually the system type - on our example, 5300 is a Sparc Station 1+ - and must be correct:
      • 5300 Sparc Station 1+
      5600 Sparc ELC
      5200 Sparc 1
      5500 Sparc 2
      80xx Classic, Sparc 4, Sparc 5 (2 digits only significant)

(see your system documentation for other values), and the last 4 digits are a serial number.

  • If your system runs licensed software, you may need to use the same hostid and/or ethernet address that was burnt into the failed EEPROM. If you don't know what the settings were - sorry, you have a problem and may need the assistance of the software supplier.
  • ^D^R means "Control D, Control R". You are NOT prompted for this after you enter the previous line; the system looks as if it has hung!
  • As of January 1997, PROM chips available are
      M48TO2-120PC1 (up to 120 MHz)
      M48TO2-150PC1 (up to 150 MHz)
      M48TO2-200PC1 (up to 200 MHz)
    which we source from Farnells, in Leeds, UK.
