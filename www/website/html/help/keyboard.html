Technical Tip

QUESTION

My workstation is fitted with an English keyboard, but I want to use software that requires a # character. How can I?

ANSWER

Run the command loadkeys /usr/share/lib/keytable/us
(or if you want a different layout, look through the files in /usr/share/lib/keytables)

It is also possible to change the keyboard layout permanently by removing the back cover from the keyboard and changing the internal switches.

For a "type 4" keyboard, the back cover panel is located under the left hand retractable foot, and is cleverly disguised! Switch settings:

NOTE

The change (UK to US) just changes £ into #, and is manageable by most people without keycap changes. Other changes (such as to and from a French keyboard) are more dramatic.

The loadkeys program changes the "streams" module which is bypassed by Open Windows, so this method works ONLY when running an unwindowed session.
